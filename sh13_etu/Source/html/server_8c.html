Detailed Description

server soutenant le jeu Sherlock Holmes. On exécute le server une fois et 4 clients afin de commencer un jeu. Ce jeu nous permet d'aborder les notions fondamentales de la programmation réseau.

Author
Arthur & Younes
Version
1.1
Date
22 mai 2019

Ce serveur propose un autre protocole que HTTP: le "Y&A" pour comprendre comment ce dernier fonctionne.
